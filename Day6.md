# Day 6: Nmap - Port Scanning Basics

## What is Nmap?
Network Mapper tool used to discover hosts and services on a network.

## Commands Used:
1. `nmap scanme.nmap.org` - Basic scan
2. `nmap -sV scanme.nmap.org` - Version detection  
3. `sudo nmap -O scanme.nmap.org` - OS detection

## Key Learning:
Open ports are entry points for attackers. SOC Analysts scan to find vulnerabilities.

## Output:
Port 22 - SSH Open
Port 80 - HTTP Open
